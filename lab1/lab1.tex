\section{Paragraph and Linebreaks}


\textbf{Part 1:} \lipsum[1]\lipsum[2] % no pararaph indentation, double linebreak causes par indentation for part 2

\textbf{Part 2:} \lipsum[1]\par\lipsum[2] % \par command creates paragraph indentation

\newpage %\pagebreak can be used as well, pagebreak acts as a newpage suggestion whereas \newpage forces a newpage at the given position.

\textbf{Part 3:} \lipsum[1]\\\lipsum[2] % linebreak but no paragraph indentation

\textbf{Part 4:} \lipsum[1]\newline\lipsum[2]\\ % same effect as \\
